# PortfolioShot

Dev screenshot beautifier. Drop a screenshot, wrap it in a device frame with a gradient background, export as PNG.

## Features

- **7 frame types:** None, macOS, Browser, iPhone, iPad, Terminal, Code editor
- **12 gradient presets** + custom color picker + angle control
- **Adjustable:** padding, corner radius, shadow intensity
- **Export scale:** 0.5x to 3x for retina/print
- **Watermark** support
- **Subtle noise texture** for premium feel
- **Paste from clipboard** (Ctrl/Cmd+V)
- **Drag and drop** anywhere on the page
- **Keyboard shortcut:** Cmd+S to export

## Usage

Open `index.html` in any browser. No build tools, no dependencies, no server.

## Tech

Single-file HTML + Canvas 2D. Zero dependencies. Fully offline.
